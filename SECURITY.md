# Security Policy

## Reporting a Vulnerability

Do not open a public GitHub issue for security vulnerabilities.

Email: security@agentmeridian.xyz or open a private GitHub advisory.

Include: description, reproduction steps, impact, and any suggested fix.
We aim to respond within 48 hours and patch within 7 days for critical issues.

## Scope

This software manages a Solana wallet with real funds. Vulnerabilities that could
lead to unauthorized transactions, private key exposure, or fund loss are critical.
